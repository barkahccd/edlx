### Metode Default: Chrome Extension

1. **Login** dulu ke https://runway.edel.finance di Chrome (wajib!)
2. Buka `chrome://extensions/`
3. Aktifkan **Developer mode** (pojok kanan atas)
4. Klik **Load unpacked** → pilih folder `edel_cookie_ext/`
5. Klik icon hijau **E** di toolbar Chrome
6. Extension otomatis baca cookie dari browser yang **sudah login**
7. Klik **Copy & Siap Kirim**
8. Langsung paste ke Telegram bot (format sudah otomatis `@acc1 edel_session=...`)
